###	<a name="intro">项目介绍</a>

该项目为LaTeX编译生成的中英文个人简历。

****

###	<a name="usage">使用说明</a>

1. 安装[MacTeX](https://tug.org/mactex/mactex-download.html)。
2. 用第一步中安装的软件`TeXShop`打开文件`resume.tex`。
3. 点击`Typeset`编译文件，即可生成中英文简历的PDF文件。
